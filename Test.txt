Buidl a simple REST API for managing students and courses using Node, Express, MongoDB

Student Collection-
name-string
age-Number
courseId-[ObjectId](Array of Courses the student is enrolled in)

Courses Collection
title- string
duration - string
instructor- string(Instructor Name)

API endpoints

1. Course Routes
	- POST /api/courses
	- GET  /api/courses
	- GET  /api/courses/:id
	- PUT  /api/courses/:id
	-DELETE /api/courses/:id

2. Student Routes
	-POST /api/students
	-GET  /api/students
	-GET  /api/students/:id
	-PUT  /api/students/:id/enroll

Optional Feature
	GET /api/stats -> show how many students are enrolled in each course
	(Hint: use $lookup and $project in aggregation)
