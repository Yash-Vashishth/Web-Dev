Task 2: Tables
1. <table> - create table
    a. border= gives it a defined border
    b. cellpadding - size of cell with info
    c. cellspacing - spaces between cells with data
2. <thead> - heading of table
3. <tbody> - body of table
    a. align - attribute to set alignment of data of cell
4. <tr>    - table row
5. <td>    - table data
    a. rowspan - stretching content of same cell over 2 rows 
    b. colspan - stretching content of same cell over 2 columns
