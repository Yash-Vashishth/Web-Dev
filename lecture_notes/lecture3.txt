5. CSS Units
    a. absolute (px,mm,cm,in)
    b. relative (vh,vw,%,rem,) ---
        vh-viewport height
        vw-viewport width
        rem- change element size w.r.t root element
        em- change element size w.r.t parent size
        eg. div{
            height: 100vh; /* 100% of viewport height */
            width: 100vw;
            font-size: 2 rem /* font size = 2x16 = 32 px*/
        }
